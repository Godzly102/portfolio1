These are some projects i have done...

Austin Market - SQL/Python

This project cleans and standardizes Austin housing data by removing outliers, normalizing numerical features, handling missing values, and standardizing text data. It also calculates useful metrics, including price per square foot and property age, to enhance analysis of real estate trends.

Netflix & Global Market

This SQL/Python script retrieves unique countries from Netflix titles, then filters for U.S.-produced titles. It also counts releases per year, lists top countries by title count, and ranks popular directors by their number of titles.

Amazon - SQL

This project transforms and analyzes Amazon product data to extract insights on top-selling companies, popular product categories, and discount trends. By parsing product names, cleaning price data, and exploring relationships between ratings and prices, it provides a clearer view of product performance across various metrics.

Cars and Fuel - Python

This Python script loads the cleaned fuel dataset and calculates estimated annual fuel costs based on city and highway MPG for rows with missing values. It groups the data by car make, calculates average MPG, fuel cost, and engine cylinder counts, and visualizes the results with a bar and line plot showing MPG, annual fuel cost, and engine cylinders by car make.

COVID-19 - SQL

This SQL script performs a comprehensive analysis of COVID-19 data, including death and vaccination statistics, across various countries and continents. It calculates death percentages, survival rates, infection rates, and vaccination coverage, while also creating temporary tables and views to track rolling vaccinations, and then visualizes the results for deeper insights into the global impact of the pandemic.

Cleaning Data - SQL

This SQL script cleans the property sales data by converting date formats, splitting address fields into individual components, and standardizing values like SoldAsVacant. It also removes duplicates and unnecessary columns to ensure the dataset is clean and ready for analysis.
